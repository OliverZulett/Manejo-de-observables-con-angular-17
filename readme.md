<h2 align="center"> Seccion 4 - Clase 1: Angular movies proyecto base </h1>

### Descripción:
Este repositorio contiene todo el avance del curso de OpenWebinars: Manejo de observables con Angular 17.

Esta rama contiene el código relacionado al avance de la clase 1 de la sección 4 en la que vimos una vista general del proyecto angular movies que trabajaremos en las siguientes secciones del curso.

### Documentación:

* [Angular CLI](https://angular.io/cli)

### Requerimientos:
Para ejecutar este proyecto necesitas tener instalado: 
* [NodeJS 20.11](https://nodejs.org/en)
* [Angular CLI](https://angular.io/cli)

### Instalación:
* acceder a la carpeta angular-movies y ejecutar:

```bash
npm install
```

### Ejecución:
* acceder a la carpeta angular-movies y ejecutar:

```bash
ng s
```
* o:
```bash
npm run start
```
